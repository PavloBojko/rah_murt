Компонент Router представляет собой следующую структуру

В корне находится конфигурационный файл Apach .htaccess который все запросы перенаправляет в папку public
в которой, находится входная точка index.php введенный url передается GET параметром на index.php.

Также в нас имеется папка model в которой находится массив наших раутов, ключом которого будет наш зарегистрированный роут а значением наша страница которая будет принадлежать нашему роуту.
[
    '/login'=>'page_login.php',
    '/register'=>'page_register.php'
]

В папке views будут находиться наши страницы.

В папке rout находится наш класс Rout который имеет :
1. свойство $spisok_url - в нем будет находиться наш массив с помощью конструктора при создании экземпляра класса мы передадим нашу предустановленную систему роутов которая находится в папке model.

2. метод add_rout($uri, $page) - добавляет роут в него два параметра первый наш uri который мы хотим зарегистрировать, а второй наша страница, которая будет соответствовать данному uri

3. метод enable() он уже можно сказать включает нашу систему роутов.
перебирает массив и сравнивает с нашим введённым uri если есть то подключает страницу соответствующею данному uri
если нет то выводит страницу 404.
